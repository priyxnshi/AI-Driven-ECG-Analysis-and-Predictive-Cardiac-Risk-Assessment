## 📊 MODEL INFORMATION

### Architecture
- Input: 256-sample ECG beat
- CNN: 2 convolutional blocks (32→64 filters)
- LSTM: 2 recurrent layers (64→32 units)
- Dense: 3 fully-connected layers (128→64→5)
- Output: 5 cardiac classes

### Training Configuration
- Epochs: 50 (or early stop)
- Batch Size: 32
- Optimizer: Adam
- Loss: Categorical Crossentropy
- Metrics: Accuracy, Precision, Recall, F1

### Classes (ECG Beat Types)
- N (Normal): Regular heartbeat
- V (Ventricular): PVC - irregular beat
- A (Atrial): PAC - premature beat
- L (LBBB): Left bundle branch block
- R (RBBB): Right bundle branch block

### Expected Performance
- Accuracy: 85-95% (on MIT-BIH)
- Precision: 0.82-0.93
- Recall: 0.85-0.92
- F1-Score: 0.83-0.92
